Datatype present in js(Primitive)
*boolean - true or false
*strings- words,sentences,mix of numbers and word.
*Number- includes integer and decimal values.
KEYWORDS ARE THREE TYPES
1.Let:
*These keywords are used to assign variables with values
*They can be declared in a line and then initialized with a value in anaother lines.
*They can be also declared and initialized in the same line
*They cannot be redeclared.....But can be reinitialized
*We cannot have Two Variables with same name in Let
2.Var:
These keywords are used to assign variables with values
They can be declared in a line and then initialized with a value in another lines.
They can be also declared and initialized in the same line
They can be redeclared.....and also can be reinitialized
We can have Two or more Variables with same name in Var
3.Const:
These keywords are used to assign variables with values
They can be declared and initialized in the same line only.
They cannot be redeclared
But const "cant" be reinitialized also
******We cannot have Two Variables with same name in const
